# Reto JS – Drum Kit

## Descripción

Reto enfocado en recrear una batería digital que reproduce sonidos al presionar teclas específicas del teclado. El objetivo es practicar eventos de teclado, manipulación del DOM y reproducción de audio en JavaScript. 

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript Vanilla

## Instrucciones para instalar y ejecutar localmente

1. Clonar o descargar este repositorio
2. Tener la carpeta `/sounds` con todos los archivos `.wav` que sean necesarios
3. Abrir `index.html` directamente en el navegador, no se requiere servidor
